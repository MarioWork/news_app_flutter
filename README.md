# Keep Up: News
This project consists of mobile application developed using flutter/dart that allows users to see the top headlines of the news filtering by country and category

## Web Api
[NewsApi](https://newsapi.org/)

## Flutter Packages
+ [http](https://pub.dev/packages/http) => For Http requests
+ [intl](https://pub.dev/packages/intl) => For date conversions
+ [url_Launcher](https://pub.dev/packages/url_launcher) => For launching articles website
+ [cached_network_image](https://pub.dev/packages/cached_network_image) => To load images from network with placeholder
+ [share_plus](https://pub.dev/packages/share_plus) => Allow user to share the article
+ [infinite_scroll_pagination](https://pub.dev/packages/infinite_scroll_pagination) => To add pagination to the app scroll 
+ [flutter_launcher_icons](https://pub.dev/packages/flutter_launcher_icons) => Add launcher icon to the app
+ [flutter_native_splash](https://pub.dev/packages/flutter_native_splash) => Customize the splash screen
